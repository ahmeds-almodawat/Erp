# v310 Enterprise Foundation Cutover

## Purpose

v310 starts the transition from a strong ERP prototype into a production-safe enterprise ERP foundation.

This version focuses on foundation, not more screens.

## Added foundation areas

- Data provider boundary
- Production configuration guard
- Permission registry
- Branch scope model
- Fiscal period model
- Posting status rules
- Activity log foundation
- Import staging foundation
- Supabase v310 migration
- Enterprise release checklist

## Professional rule

The frontend is the interface.

The backend is the authority.

The database is the truth.

## What should happen next

v311 should focus on server-side posting functions:

- Manual journal posting
- Opening balance posting
- Purchase invoice posting
- Supplier payment posting
- Sales/POS batch posting
- Inventory adjustment posting
- Reversal posting
- Posting validation RPCs
