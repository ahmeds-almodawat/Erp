# v41 Inventory Count Sheets

Adds a professional monthly stock count workflow:

1. Export a system-generated count sheet by store.
2. Physical team fills counted_qty.
3. Upload the count sheet.
4. ERP compares counted quantity with system quantity.
5. Shortage/surplus becomes an approval request.
6. Approval posts inventory movement and GL entry.

Shortage: Dr Inventory Variance / Cr Inventory.
Surplus: Dr Inventory / Cr Inventory Variance.
