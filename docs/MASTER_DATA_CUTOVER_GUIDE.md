# Master Data Cutover Guide

Recommended sequence:

1. Branches
2. Stores
3. Chart of accounts
4. Item categories
5. Suppliers
6. Items
7. Opening balances and opening stock later

Important:

- Codes must be unique.
- Arabic and English names should be complete.
- Parent chart accounts should not allow posting.
- Items should use weighted average cost for production pilot.
- Do not delete master data after transactions exist. Archive instead.
