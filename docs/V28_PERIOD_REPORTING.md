# V28 Period Reporting

## What changed
Reports are now period-aware instead of showing all data only.

## Period controls
The Reports module now includes:

- All periods
- Today
- Current month
- Current quarter
- Current year
- Custom date range

## Reporting behavior

- Net sales, COGS, gross profit, supplier spend, menu engineering, and journal summaries use the selected period.
- Inventory valuation uses movements up to the selected end date, so it behaves like an as-of stock valuation.
- CSV exports include the selected period or as-of date.

## Why this matters
A professional ERP must let management review the business by period: daily, monthly, quarterly, yearly, or custom dates. This is required for branch P&L, VAT, AP, inventory valuation, and management reporting.

## Remaining future upgrades

- Comparative reports: current period vs previous period.
- Budget vs actual by period.
- Monthly branch P&L matrix.
- PDF/Excel report pack.
- Backend fiscal-period integration with locked periods.
