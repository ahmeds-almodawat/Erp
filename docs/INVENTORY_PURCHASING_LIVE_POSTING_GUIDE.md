# Inventory and Purchasing Live Posting Guide

Recommended sequence:

1. Confirm v329-v334 backend gate passed.
2. Confirm v335-v339 master data and GL gate passed.
3. Stage opening stock.
4. Post opening stock.
5. Test purchase receipt.
6. Test purchase invoice.
7. Test supplier payment.
8. Test stock count variance.
9. Reconcile stock balances, AP, VAT, and GL before production go-live.

Never delete posted stock or finance records. Use reversal/correction flows.
