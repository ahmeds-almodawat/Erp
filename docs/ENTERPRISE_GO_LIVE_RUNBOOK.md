# Enterprise Go-Live Runbook

Before production:

1. Run QA all.
2. Apply migrations to staging.
3. Verify RLS tests.
4. Test backup and restore.
5. Load UAT data.
6. Complete UAT scenarios.
7. Confirm admin users and roles.
8. Post opening balances.
9. Post opening stock.
10. Open first fiscal period.
11. Confirm support and audit logging.
12. Approve production deployment.
